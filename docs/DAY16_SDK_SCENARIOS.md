# DAY16: SDK Scenarios (Finance Ops)

本文对应 `IMPL-16-05`，给出 5 个财务运维场景示例（SDK 侧）。

## 场景 1：费用报销流程配置

- 输入：审批链配置（节点、角色、阈值）
- SDK：业务字段校验 -> 映射为通用 IR
- 输出：提交到 attach 事务链

## 场景 2：总账科目映射配置

- 输入：科目映射表、默认科目策略
- SDK：语义校验（重复映射、空科目） -> 通用结构
- 输出：统一版本化提交

## 场景 3：资金结算审批链配置

- 输入：结算级别、授权矩阵
- SDK：规则归一化 -> IR
- 输出：提交 + watch 审计事件

## 场景 4：月结参数切换

- 输入：月结窗口参数、生效时点
- SDK：时间参数校验与兼容转换
- 输出：批次提交并保留审计链路

## 场景 5：多账套动态切换配置

- 输入：账套路由与优先级
- SDK：路由规则映射与冲突检测
- 输出：按账套 URI 粒度提交
